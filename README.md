## 🧠 实验心得 (Lessons Learned)

在 AI 辅助编码的过程中，我总结出的三条准则：

1.  **Context is King (上下文至上):** 提供准确的业务背景比长篇累牍的指令更有效。
2.  **Verify, Don't Just Trust (验证而非盲从):** 始终保持 Code Review 的习惯，人类负责最后的底线。
3.  **Iteration Speed (迭代速度):** 利用 AI 快速失败，快速修正，这是传统开发模式无法比拟的优势。
