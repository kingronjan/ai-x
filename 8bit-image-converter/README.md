# 8-Bit 图像转换器

这是一个简单的 Web 应用，允许用户上传图片并将其转换为 8-bit 风格，然后预览并下载转换后的图片。

## 功能

*   上传图片文件。
*   将上传的图片转换为 8-bit 风格。
*   实时预览原始图片和 8-bit 转换后的图片。
*   可调节的色彩分辨率（2-256），以控制 8-bit 效果的强度。
*   下载转换后的 8-bit 图片。

## 如何使用

1.  **克隆或下载项目：**
    如果您从 Git 仓库获取此项目，请克隆它：
    ```bash
    git clone [repository-url]
    cd 8bit-image-converter
    ```
    如果您是直接下载文件，请确保 `index.html`、`style.css` 和 `script.js` 文件都在同一个 `8bit-image-converter` 文件夹中。

2.  **打开应用：**
    在您的文件浏览器中，找到 `8bit-image-converter` 文件夹，然后双击 `index.html` 文件。它将会在您的默认网页浏览器中打开。

3.  **上传图片：**
    在网页上，点击 "点击或拖拽图片到这里" 区域，或者直接将图片文件拖拽到该区域。

4.  **调整色彩分辨率：**
    上传图片后，您可以使用 "色彩数量" 滑块来调整 8-bit 效果的强度。色彩数量越少，8-bit 效果越明显。

5.  **预览图片：**
    网页上会显示 "原始图片" 和 "8-Bit 效果" 的预览。

6.  **下载图片：**
    当您对 8-bit 效果满意时，点击 "下载图片" 按钮即可将转换后的图片保存到您的设备。

## 文件结构

*   `index.html`: 应用程序的 HTML 结构。
*   `style.css`: 应用程序的样式表。
*   `script.js`: 应用程序的 JavaScript 逻辑，处理图片上传、转换和下载。

## 技术栈

*   HTML5
*   CSS3
*   JavaScript (Canvas API)
